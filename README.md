# Homebrew Tap for Desktop Waifu

This is the official Homebrew tap for [Desktop Waifu](https://github.com/yv-was-taken/desktop-waifu), a desktop companion app featuring animated 3D VRM characters with AI-powered conversational chat.

## Installation

```bash
brew tap yv-was-taken/desktop-waifu
brew install desktop-waifu
```

## Updating

```bash
brew update
brew upgrade desktop-waifu
```

## Uninstalling

```bash
brew uninstall desktop-waifu
brew untap yv-was-taken/desktop-waifu
```

## Platform Support

| Platform | Mode |
|----------|------|
| macOS | Window mode |
| Linux (Wayland) | Full overlay support |
| Linux (X11) | Window mode |

## Issues

For issues with the formula, please open an issue in this repository.

For issues with the application itself, please open an issue at [desktop-waifu](https://github.com/yv-was-taken/desktop-waifu/issues).
